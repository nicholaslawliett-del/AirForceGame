## 📖 Tentang Proyek
Game ini dibuat dan disusun oleh Kelompok 8 yang bertujuan sebagai salah satu tugas wajib pada mata kuliah Komputer Grafik.

## 🛠 Teknologi yang Digunakan
- [JavaScript]
- [HTML5 Canvas / Library Game 2D]
- [Tools lainnya: VS Code, Git, GitHub]

## ✨ Fitur Utama
- [ ] Fitur 1: Pergerakan pesawat yang responsif dengan sprite custom.
- [ ] Fitur 2: Sistem skor dan manajemen nyawa pemain (heart system).
- [ ] Fitur 3: Efek suara (SFX) ledakan dan tembakan yang imersif.

## 🚀 Cara Instalasi
1. Clone repositori ini:
   `git clone https://github.com/nicholaslawliett-del/AirForceGame.git`
2. Masuk ke folder proyek.
3. Buka file `index.html` pada browser Anda untuk menjalankan game.

## 🤝 Kontribusi
Silakan buat pull request jika ingin menambahkan fitur baru atau memperbaiki *bug*. Pastikan untuk mengikuti struktur folder `assets/` dan `docs/` yang telah ditentukan.

  - `player_heart.png`: Ikon nyawa pemain

- **Suara/SFX (`assets/sounds/`):**
